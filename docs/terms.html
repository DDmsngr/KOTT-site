Пользовательское соглашение

Редакция от 15 апреля 2026 г.

Настоящее Пользовательское соглашение регулирует отношения между пользователем и разработчиком мобильного приложения КОТТ. Устанавливая и используя Приложение, вы принимаете условия настоящего соглашения в полном объёме.

1. Предмет соглашения

Настоящее Пользовательское соглашение (далее — Соглашение) заключается между пользователем и разработчиком мобильного приложения «КОТТ — Книга Описи Творческих Товаров» (далее — Приложение).

Приложение предназначено для ведения учёта произведений искусства: регистрации работ, хранения фотографий, отслеживания выставок и продаж, генерации документов.

2. Условия использования

Пользователь вправе:

  • Использовать Приложение для личного учёта произведений искусства
  • Создавать и хранить данные о своих произведениях
  • Экспортировать и импортировать свои данные
  • Генерировать документы на основе внесённых данных

Пользователь обязуется:

  • Не использовать Приложение в противоправных целях
  • Не вносить заведомо ложные сведения
  • Соблюдать авторские права при использовании изображений

3. Интеллектуальная собственность

Все права на Приложение, включая исходный код, дизайн и документацию, принадлежат разработчику. Пользователю предоставляется неисключительная лицензия на использование Приложения.

Все данные, внесённые пользователем (описания произведений, фотографии, документы), являются собственностью пользователя. Разработчик не претендует на права на контент пользователя.

4. Версии приложения

Приложение распространяется в двух версиях:

  • Бесплатная версия — ограниченное количество произведений, все базовые функции
  • Полная версия — неограниченное количество произведений, все функции. Приобретается единоразово, без подписки

Разовая оплата предоставляет бессрочный доступ к полной версии Приложения на одном устройстве.

5. Данные пользователя

Все данные хранятся локально на устройстве пользователя. Разработчик не имеет доступа к данным пользователя. Пользователь несёт ответственность за сохранность своих данных и создание резервных копий.

При удалении Приложения все локально хранящиеся данные удаляются безвозвратно. Рекомендуем создавать резервные копии через раздел «Настройки → Экспорт данных».

6. Генерируемые документы

Документы, генерируемые Приложением (сертификаты подлинности, договоры купли-продажи, акты передачи), носят справочный характер. Разработчик не несёт ответственности за юридическую силу документов в конкретных правоотношениях. Для юридически значимых сделок рекомендуется консультация с юристом.

7. Ограничение ответственности

Приложение предоставляется «как есть». Разработчик не гарантирует бесперебойную работу Приложения и не несёт ответственности за:

  • Потерю данных вследствие неисправности устройства
  • Убытки, возникшие вследствие использования Приложения
  • Действия третьих лиц, получивших доступ к устройству пользователя
